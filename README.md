# Vitest Unit Testing

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Running Tests with Vitest
This project uses [Vitest](https://vitest.dev/) for unit testing.

### Run Tests
```sh
npm run test
```
